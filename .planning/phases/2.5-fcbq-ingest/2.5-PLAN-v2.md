# Phase 2.5 v2 — Execution Plan (basquethero.cat)

## Why v2

v1 PLAN (`2.5-PLAN-v1-archived.md`) assumed FCBQ (`basquetcatala.cat`) as data source. Wave 0 researcher spike (commit `3b56f3f`) found **reCAPTCHA v3 wall + JPG actas + JWT RS256 microservice** → violated D2.5-02 (httpx-only) + D2.5-09 (zero new deps). **STOP**.

AFK overnight cluster C8 (commit `ee40024`) ran a v2 spike against `basquethero.cat`: robots permissiu, sitemap 17K URLs, Next.js+RSC streamed via `self.__next_f.push`, slugs directes `/liga/cc-2a-m-grup-{01..06}`. Stdlib parser feasible (urllib + re + json, ~200-400 línies). Spike `spike-basquethero/spike.py` (130 línies) demostra extracció funcional. **PROCEED**.

### Task 7 — Write load_basquethero.py

**Issue:** `enhancement` + `P1` + `phase/2.5-ingest-cli` — "Write data/seed/load_basquethero.py UPSERT loader (D2.5-06)"
**File:** `data/seed/load_basquethero.py`
**Pattern:** Mirror `src/basketball_stats/seed/minimal.py`. argparse `--force`, `get_session_factory()`, idempotent. **UPSERT amb `sqlalchemy.dialects.postgresql.insert(...).on_conflict_do_update(...)`** per D2.5-06.

**Logic:**
1. Argparse: positional `fixture_path`, optional `--dry-run`, `--verbose`.
2. Read JSON, parse amb `BasqueteroFixture` (raises on schema mismatch).
3. Transacció única:
   - UPSERT competition (`ON CONFLICT (slug, season)`)
   - UPSERT clubs (derived from teams, `ON CONFLICT (slug)`)
   - UPSERT teams (`ON CONFLICT (basquethero_team_id)`)
   - UPSERT players (`ON CONFLICT (license_id)` — només jugadors amb license_id, els altres skip + warning)
   - UPSERT rosters (`ON CONFLICT (player_id, team_id, season)`)
   - UPSERT games (`ON CONFLICT (basquethero_game_id)`)
   - UPSERT box_scores (`ON CONFLICT (game_id, player_id)`)
4. Print summary: rows inserted vs updated vs unchanged + skipped (jugadors sense license).

**Crucial:** Loader escriu només inputs a GENERATED columns (`val`, `reb`). Postgres computa.

### Task 9 — Implement client.py (urllib + rate limit + retry stdlib)

**File:** `src/basketball_stats/ingest/basquethero/client.py`
**Surface:**
- `class BasqueteroClient(base_url, rate_limit_seconds=1.0, jitter_range=(0.0, 0.5), user_agents=DEFAULT_UA_POOL, max_retries=5)`
- `def fetch(self, path: str) -> bytes` — síncron (NO async — D2.5-02 relaxed). Returns raw response body.

- Internal: `time.sleep(rate_limit_seconds + random.uniform(*jitter_range))` entre requests.
- Internal: rota UA per request d'un pool de 4-5 Chrome/Firefox UAs (NO googlebot).
- Raises `BasqueteroFetchError(url, status, attempt)` on terminal failure.
